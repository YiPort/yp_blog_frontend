## 如何运行


### 安装依赖
>npm install

### 启动服务
默认端口8080
>npm run dev



## 项目特点
- 采用 Restful 风格的 API，注释完善，代码遵循阿里巴巴开发规范，有利于开发者学习
- 为解决多个子系统内代码大量重复的问题，抽象模型层和业务层代码为公共模块

## 技术介绍
Spring Boot + MyBatis-Plus + Mysql + Redis + vue

## 运行环境

**服务器环境：** CentOS7.6
